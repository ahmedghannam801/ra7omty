# Ra7maa 🌹

A personal romantic web experience.

## Live Site
After enabling GitHub Pages, the site will be available at:
`https://YOUR-USERNAME.github.io/ra7maa`

## How to enable GitHub Pages
1. Go to your repository **Settings**
2. Click **Pages** from the left menu
3. Under **Source** → select **main** branch → **/root** folder
4. Click **Save**
5. Wait 1-2 minutes → your link is ready 💕
